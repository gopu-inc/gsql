Documentation

Module Database

Gestion des connexions, transactions et opérations principales.

Exemple d'utilisation Python 
from gsql.database import Database

# Création d'une instance
db = Database("myapp", storage_type="yaml")

# Exécution de requête
results = db.execute("""
    SELECT * FROM users 
    WHERE age > ? AND status = ?
""", [18, "active"])

print(f"{len(results)} utilisateurs trouvés")

Module Parser

Analyse syntaxique SQL et validation des requêtes.

Exemple de parsing Python 
from gsql.parser import SQLParser

parser = SQLParser()
ast = parser.parse("SELECT * FROM users")

if ast.validate():
    print("Requête valide")
    print(f"Tables: {ast.get_tables()}")

Module Storage

Système de stockage persistant multi-format.

Exemple de stockage Python 
from gsql.storage import StorageEngine

storage = StorageEngine(backend="yaml")
storage.save("data.yml", {"users": [...]})

data = storage.load("data.yml")
print(f"Données chargées: {len(data['users'])}")

Module NLP

Traitement du langage naturel pour requêtes.

Exemple NLP Python 
from gsql.nlp.translator import NLTranslator

translator = NLTranslator(language="fr")
sql = translator.translate("Affiche les produits populaires")

print(f"SQL généré: {sql}")
print(f"Confiance: {translator.get_confidence():.1%}")

Référence API

POST /api/database/create

Crée une nouvelle base de données GSQL.

{
  "name": "string",
  "storage_type": "yaml|json|binary"
}
GET /api/database/{name}/query

Exécute une requête SQL.

{
  "q": "SELECT * FROM users",
  "format": "json"
}
POST /api/nlp/translate

Traduit le langage naturel en SQL.

{
  "text": "Affiche les utilisateurs actifs",
  "language": "fr"
}

Documentation API

POST /api/database/create

Créer une nouvelle base de données

{
  "name": "string",
  "storage_type": "yaml|json|binary",
  "config": {...}
}
GET /api/database/{name}/query

Exécuter une requête SQL

{
  "query": "SELECT * FROM users",
  "params": [...]
}
POST /api/nlp/translate

Traduire du langage naturel en SQL

{
  "text": "Affiche les utilisateurs de plus de 25 ans",
  "language": "fr"
}
